取り敢えずインストールする

仮想環境作成

まずはpythonの仮想環境をつくります.
pythonはインストールしておきます.

mkdir ~/programs/mine/Tensorflow
# 実際は好きなフォルダでいいです
cd ~/programs/mine/Tensorflow
python -m venv .venv

仮想環境を使います.

source .venv/bin/activate

インストール

最適化していないものでいいなら, インストールは簡単です.

pip install tensorflow

しばらくこれでやっていましたが,ノートPCではスペック的にしんどそうだったので、マシンに合わせ、最適化していきました.
